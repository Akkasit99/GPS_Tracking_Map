GPS Tracking — Modular Version
แอป GUI สำหรับตรวจสอบเส้นทางและสถานะการขับเคลื่อนจากไฟล์ CSV โดยใช้ CustomTkinter และ TkinterMapView สามารถเลือกช่วงเวลา กำหนดจำนวนจุดที่จะวาด เปลี่ยนสไตล์แผนที่ ซูมไปที่ตำแหน่งรถ ดูระยะทางรวม และประวัติสถานะได้แบบเรียลไทม์

✨ ฟีเจอร์หลัก (Features)
รองรับ Drag & Drop: สามารถลากไฟล์ CSV มาวางลงในหน้าต่างโปรแกรมเพื่อโหลดข้อมูลได้ทันที

ระบบสองภาษา (Bilingual): รองรับการสลับภาษาของ UI ระหว่าง ภาษาไทย (TH) และ ภาษาอังกฤษ (EN)

เครื่องมือวัดระยะทาง (Measurement Tool): คลิกบนแผนที่เพื่อวาดเส้นและวัดระยะทางระหว่างจุด (รองรับหน่วย กิโลเมตร/เมตร และ ไมล์/ฟุต)

สรุประยะทางรวม (Total Distance): คำนวณระยะทางของเส้นทางจริง (Trajectory) ด้วยสูตร Haversine อย่างแม่นยำ

เข็มทิศชี้เป้านอกจอ (Off-screen Indicator): เมื่อรถวิ่งออกนอกจอ จะมีลูกศรสีแดงชี้บอกทิศทางและระยะห่าง พร้อมคลิกเพื่อซูมตามรถได้

รองรับแผนที่หลากหลาย: OpenStreetMap, Google Normal, Google Satellite และเพิ่ม Amap (China) สำหรับแสดงพิกัดในจีนให้ตรงถนน

รองรับไฟล์จากจีน: อ่านชื่อคอลัมน์และสถานะเครื่องยนต์ที่เป็นภาษาจีนได้อัตโนมัติ

ตัวกรองข้อมูล: เลือกดูเฉพาะช่วงวันที่ (All Days หรือระบุวัน), ช่วงเวลา, ช่วงแถว (Start–End) และจำกัดจำนวนจุด (Limit points) เพื่อลดความหน่วง

ระบบสถานะรถ: แสดงจุดเริ่ม (Start), จุดสิ้นสุด (End) และสถานะรถ (Running / Idling / Engine Off) พร้อมไอคอนแยกสีชัดเจน

📂 โครงสร้างโปรเจค
main.py จุดเริ่มต้นและ UI หลักของแอป

utils.py โหลดไอคอนรถ, ประมวลผลไฟล์ CSV, และระบบคำนวณคณิตศาสตร์ (Haversine)

components.py วิดเจ็ตสเกลบาร์แบบโปร่งใส

config.py ตั้งค่าธีม สี และค่าทั่วไปของระบบ

map_cache.db (สร้างอัตโนมัติ) สำหรับแคชภาพแผนที่แบบออฟไลน์

💻 ข้อกำหนดระบบ
แนะนำ Python 3.10+

🛠 การติดตั้งแพ็กเกจ (Installation)
ติดตั้งไลบรารีที่จำเป็นทั้งหมดด้วยคำสั่ง:

Bash
pip install customtkinter tkintermapview pandas pillow tkinterdnd2
หมายเหตุ:

หากรันบน Virtual Environment (.venv) ต้องมั่นใจว่าติดตั้งแพ็กเกจลงใน .venv เรียบร้อยแล้ว

บางสภาพแวดล้อมอาจต้องการ darkdetect (ใช้ร่วมกับ customtkinter) ให้ติดตั้งเพิ่มด้วย pip install darkdetect

🚀 วิธีใช้งาน (Usage)
รันแอปด้วยคำสั่ง:

Bash
python main.py
โหลดไฟล์: กดปุ่ม “เลือกไฟล์ CSV” หรือ ลากไฟล์ CSV มาวางลงในหน้าต่างโปรแกรม

กรองข้อมูล: เลือกช่วงวันที่ เวลา แถวข้อมูล หรือจำนวนจุดที่ต้องการแสดง

ตั้งค่าแผนที่: เลือกสไตล์แผนที่ได้จากเมนูมุมขวาล่าง (หากดูข้อมูลในจีน แนะนำให้ใช้ Amap (China))

กด “Apply Settings” (นำไปใช้) เพื่อประมวลผล

ใช้สไลเดอร์ Timeline ด้านล่างเพื่อเลื่อนดูแอนิเมชันการเดินทางของรถตามเวลาจริง

วัดระยะทาง: กดปุ่ม "📏 เริ่มวัดระยะทาง" แล้วคลิกซ้ายบนแผนที่เพื่อมาร์คจุด (คลิกขวาเพื่อยกเลิกจุดล่าสุด)

📄 รูปแบบไฟล์ CSV ที่รองรับ
โปรแกรมออกแบบมาให้ยืดหยุ่น โดยจะค้นหาชื่อคอลัมน์เหล่านี้อัตโนมัติ (ไม่สนตัวพิมพ์เล็ก/ใหญ่):

พิกัดละติจูด: lat, latitude, 纬度

พิกัดลองจิจูด: long, lon, lng, longitude, 经度

เวลา: r-time, time, timestamp, 设备时间 (ระบบจะพยายามแปลงเป็น datetime ให้อัตโนมัติ)

ความเร็ว (เสริม): gps_speed, speed, 速度(km/h)

สถานะเครื่องยนต์ (เสริม): acc-on, acc, 部件状态 (ระบบจะถือว่ารถสตาร์ทหากพบคำว่า "1", "ON", หรือ "引擎开")

การเข้ารหัสไฟล์: พยายามอ่านเป็น utf-8 เป็นหลัก หากล้มเหลวจะลองอ่านแบบ cp874 (สำหรับภาษาไทยเก่า)

⌨️ คีย์ลัด (Shortcuts)
Left Arrow : เลื่อนไทม์ไลน์ย้อนหลัง

Right Arrow : เลื่อนไทม์ไลน์ไปข้างหน้า

ขณะอยู่ในโหมดวัดระยะทาง:

Delete หรือ Backspace : ลบจุดวัดระยะทางล่าสุด (Undo)

Escape (Esc) : ล้างเส้นวัดระยะทางทั้งหมดบนหน้าจอ

💡 ทิปเพิ่มเติม
หากมีไฟล์ car.png ในโฟลเดอร์เดียวกับโปรเจค แอปจะใช้เป็นไอคอนรถอัตโนมัติ (หากไม่มี ระบบจะสร้างไอคอนเวคเตอร์สำรองให้)

หากนำเข้าข้อมูลที่มีขนาดใหญ่มาก (ระดับแสนบรรทัด) การจำกัด "จำนวนจำกัดจุดแสดงผล (Limit Points)" จะช่วยให้โปรแกรมไม่ค้างและตอบสนองได้ลื่นไหลขึ้น

ระยะทางรวมที่แสดงในหน้าต่างด้านซ้าย คือระยะทางที่ถูกคำนวณบวกทบกันตลอดเส้นทาง (Cumulative Haversine Distance) ซึ่งมีความแม่นยำสูง
